# 📂 Carpeta de datos

Este directorio está destinado a almacenar el dataset utilizado en el análisis principal del proyecto:

- **Archivo:** `GCB2022v27_MtCO2_flat.csv`  
- **Descripción:** Contiene las emisiones anuales de CO₂ (en millones de toneladas) por país y tipo de fuente, según los registros del Global Carbon Project (GCP).  
- **Periodo cubierto:** 1750–2021  
- **Columnas principales:** `Country`, `Year`, `Total`, `Coal`, `Oil`, `Gas`, `Cement`, `Flaring`.

🔗 **Fuente oficial del dataset:**  
[Global Carbon Project – CO₂ Emissions Dataset](https://www.globalcarbonproject.org/carbonbudget)

⚠️ *Por razones de tamaño y derechos de distribución, el archivo no se incluye directamente en este repositorio.*  
Para reproducir el análisis, descarga el dataset desde la fuente oficial y colócalo en esta carpeta.
